
# Project Overview

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Installation
```bash
npm install
npm run dev
```

## Features
- AI-powered chat interface
- Responsive design
- Analytics dashboard
- Secure authentication

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Supabase
- Gemini AI

## Development
- Run tests: `npm test`
- Build project: `npm run build`
